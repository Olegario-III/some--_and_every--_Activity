# Some() and Every() Activity

A simple interactive web app that demonstrates the use of JavaScript's **`Array.some()`** and **`Array.every()`** methods using a student grading example.

## 📌 Overview

This project checks whether **some students passed** or **all students passed** based on their scores. It showcases the practical difference between the `some()` and `every()` array methods.

## ✨ Features

- Clean and simple user interface
- Real-time result display
- Demonstrates `Array.some()` and `Array.every()`
- Responsive design

## 🛠 Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## 📁 Project Structure
├── index.html
├── Styles.css
└── script.js

## 🚀 How to Run

1. Download or clone the project
2. Make sure all three files are in the same folder:
   - `index.html`
   - `Styles.css`
   - `script.js`
3. Open `index.html` in your browser

## 📋 What It Does

The application:
- Displays a list of students with their scores
- Has buttons to check:
  - **"Did some students pass?"** → Uses `.some()`
  - **"Did all students pass?"** → Uses `.every()`
- Shows the result instantly
